# 🛡️ Kejasafe

**Kejasafe** is a robust and secure application platform built for efficiency and trust. Our mission is to provide an integrated solution that balances top-tier security with a seamless user experience.

---

## 🏛️ Architecture & Project Structure

This project is maintained as a **Monorepo**, separating the concerns of the backend logic from the frontend presentation to ensure scalability and ease of maintenance.

- [**`api/`**](./api) - The Backend API powered by **Laravel**. This acts as the central intelligence of Kejasafe, handling business logic, database orchestration, and authentication.
- [**`web/`**](./web) - The Frontend Web Application built with **Next.js**. It provides a modern, fast, and responsive interface for end-users.

## 🚀 Quick Start Guide

To get a local development environment running, follow these high-level steps:

1. **Backend Setup**:
   Navigate to the `api/` directory and follow its README for installation and environment configuration.
2. **Frontend Setup**:
   Navigate to the `web/` directory and follow its README for package installation and dev server execution.

## 🛠️ Technology Stack

- **Backend**: Laravel
- **Frontend**: Next.js
- **Styling**: Tailwind CSS
- **Database**: SQL-driven
- **Version Control**: Git

## 🔒 Security & Support

Security is at the heart of Kejasafe. If you discover any security vulnerabilities, please do not disclose them publicly. Reach out to our security team via official channels.
